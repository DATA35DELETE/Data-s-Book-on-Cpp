Karar Yapıları

Karar Yapıları Nedir?

Karar yapıları, programda koşula bağlı olarak akışı değiştiren ve farklı durumlar için farklı işlevler veya kodlar çalıştırabilceğimiz bir yapıdır.
Program akışı, bir programın kodları okuma sırasına verilen isimdir. C++, kodları yukarıdan aşağı doğru okumakta ve program akışının başlangıcı Main kısmıdır. Main ilk kodların çalıştığı kısımdır.
Peki neden karar yapılarını kullanırız? Çünkü farklı durumlar farklı ihtiyaçlar doğrur! Açıklamak gerekirse, Bir ATM yazılımı yazdığımızı düşünelim. Bu yazılımda kullanıcı kartını kart bölmesine yerleştiricek ve makine şifre istiyecektir. Kullanıcı şifresini girdiğinde makine, şifresinin doğru olup olmadığını kontrol etmesi lazım ki kullanıcının gerçekten o kartı sahibi olduğunu algılasın ve işlemleri başlatsın. İşte tam bu kısımda kontrol yapıları kullanılır. Kontrol yapısı kullanıcının girdiği şifreyle banka şirketinin veritabanlarındaki o karta atanmış şifreyi karşılaştırır. Eğer şifre eşleşiyorsa işlemleri başlatır, eşleşmese ise uyarı mesajı verip işlemi iptal eder. İşte kontrol yapıları sizin paranızdan tutunda oyunlardaki düşman öldürme mekaniğinden yapay zekayı mümkün kılan algoritmalara kadar her alanda vardır.
Kontrol yapıları algoritma kurmanın anahtarıdır. Algoritma almadan bir sistemin çalışması iplik kadar zayıf bir ihtimal olduğunu algoritmalar önemli yapılardır. Bu sebeble iyi algoritma kurmak için kontrol yapılarını ismimiz gibi kavramalı ve işlem yapabilme kapasitemizi artırmalıyız.

If-Else If-Else

Bu yapı C++ dilinde bir kontrol yapısıdır.

If

If, parantezlerindeki değer True ise kendi kod bloğundaki kodları çalıştırır.
Şimdi örnek olması açısından bir kullanıcı tarafından akışı değiştirme programı yazalım. Kullanıcı ilk programı çalıştırdığında bir girdi istiyelim. Eğer biri seçerse "Birinci yolu terci ettiniz" diyen bir kod satırları yazalım. Ayrıca programın sonlarına doğruda "Yol bitti" diyen bir kod daha yazalım ki programın bittiğini algılayalım.

#include <iostream>

using namespace std;

int main()
{
    cout << "Birinci yoldan gidicek misiniz?" << endl << "-->"; // Kullanıcıdan görsel olarak giriş isteme ve tercih kısmı

    int secim1; // Kullanıcının girdisinin kayıt edilceği değişken
    cin >> secim1; // Program tarafında kullanıcıdan girdi istemek

    if (secim1 == 1) // Eğer kullanıcının girdiği veri 1 ise
    { // Bu parantezlerdeki kodu çalıştır
        cout << endl << "Birinci yolu terci ettiniz..." << endl; // Birinci yolu tercih ettiğini bildirmek
    }

    cout << endl << "Yol bitti" << endl; // Programın bitiğine dağir mesajı verelim

    return 0;
}

Çıktısı ise;
Gördüğünüz gibi seçimlere göre programın akışını değiştirdik.

Else If

Else If, If çalışmaz ve kendi parantezlerindeki değer True ise kod bloklarını çalıştırır.
Else If, kendi başına kullanılamaz ama bir If'ın altına istenilen miktar kadar yazılabilir.
Şimdi gelin örneğimizi geliştirilem ve ikinci bir yok ekliyelim.

#include <iostream>

using namespace std;

int main()
{
    cout << "Yol tercihi programı" << endl;
    cout << "1: Birinci yoldan gider" << endl; // Yol terci menüsü güncelledik
    cout << "2: Ikinci yoldon gider" << endl;

    cout << "-->";

    int secim1;
    cin >> secim1;

    if (secim1 == 1)
    {
        cout << endl << "Birinci yolu tercih ettiniz..." << endl;
    } 
    else if(secim1 == 2) // Eğer kullanıcının girdiği değer 2 ise
    { // Bu komut bloğunu çalıştır
        cout << endl << "Ikinci yolu tercih ettiniz..." << endl; // İkinci yoldan gidildiğini bildir
    }

    cout << endl << "Yol bitti" << endl;

    return 0;
}


Çıktısı ise;

Else

Else, If bloğu ve Else If blokları çalışmazsa koşulsuz olarak kendisini çalıştırır.
Programı daha fazla geliştirmek adına birde Else ekliyelim. Else'ın görevi ise kullanıcı seçimler dışı bir girdi girdiğinde uyarı vererek progam akşını etkilesin.

#include <iostream>

using namespace std;

int main()
{
    cout << "Yol tercihi programi" << endl;
    cout << "1: Birinci yoldan gider" << endl;
    cout << "2: Ikinci yoldon gider" << endl;

    cout << "-->";

    int secim1;
    cin >> secim1;

    if (secim1 == 1)
    {
        cout << endl << "Birinci yolu tercih ettiniz..." << endl;
    } 
    else if(secim1 == 2)
    {
        cout << endl << "Ikinci yolu tercih ettiniz..." << endl;
    }
    else // Eğer If ve Else If'ler çalışmaz ise
    { // Bu kod bloklarını çalıştır
        cout << endl << "Hicbir yol tercih edilmedi..." << endl;
    }

    cout << endl << "Yol bitti" << endl;

    return 0;
}


Çıktı;

Switch-Case-Default

Switch-Case-Default, C++ yazılım program dilindeki bir başka bir kontrol yapısıdır.
Switch, kontrol edilecek olan değişkeni Switch'ın parantezlerine yazılır ve o değişkeni kontrol edicek Case'ler Switch'in kod bloklarına yazılır.
Case, Switch'in parantezlerindeki değişkenini kontrol edicek olan yapıdır. Case'in kontrolu sonucu çalışıcak olan tüm kodların satırlarının en son satırına break deyimi yazılmalıdır. Eğer yazılmasza Case'ler beklenemedik davranışlar sergileyebilir. Case'lerin tanımlanna değişkenin değerini kontrol etmek için case degiskeninOlmasıGerekDegeri: şeklinde veri kontrol edilir.
Default ise tüm Case'ler çalışmaz ise kendi kod kısmını çalıştıran bu kontrol yapısının son kontrol elemanıdır. Bir nevi If-Else If-Else yapısındaki Else gibi çalışır.
Şimdi ise If-Else If-Else örneğimizi Switch-Case-Default yapısına uyarlayalım.

#include <iostream>

using namespace std;

int main()
{
    cout << "Yol tercihi programi" << endl;
    cout << "1: Birinci yoldan gider" << endl;
    cout << "2: Ikinci yoldon gider" << endl;

    cout << "-->";

    int secim1;
    cin >> secim1;

    switch (secim1) // Kontrol edilecek değişkeni parantezlere yazdık
    {
        case 1: // Eğer değeri 1 ise
            cout << endl << "Birinci yolu tercih ettiniz..." << endl; // Bu komutu çalıştır
            break; // Case1'in sonu

        case 2: // Eğer değeri 2 ise
            cout << endl << "Ikinci yolu tercih ettiniz..." << endl; // Bu komutu çalıştır
            break; // Case2'in sonu
        
        default: // Eğer Case'lerin hiç biri çalışmaz ise
            cout << endl << "Hicbir yol tercih edilmedi..." << endl; // Bu komutu çalıştır
            break; // Case2'in sonu
    }

    cout << endl << "Yol bitti" << endl;

    return 0;
}
