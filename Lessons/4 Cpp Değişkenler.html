Değişkenler

Değişkenler Nedir?

Değişkenler, yazılımlarda verileri geçici olarak saklamamızı sağlayan yapılardır. Değişkenlerin depolandığı donanımlar ise RAM ve işlemcinin içerisindeki Registers'larda depolanır.
Değişkenler bizler için not defterleri gibi çalışır. Daha sonradan kullanıcağımız veriler bir değişken ismine atıyarak kullanıcağımız zamanlarda değişkeni çağrarak içerisindeki değerli kullanabiliriz. İstersekde değişkenin değerini değiştirebiliriz.

Sayısal Değişkenler

Bilgisayarların yapıları gereği bir çok sayı türü vardır.

Short

Short, C++ yazılım dilinde sayısal değişkenlerde en küçük sayı aralığına sahip değişken türüdür. -32,768 ila 32,767 sayılarının arasında bir değer alabilir. RAM'de Short değişken türü iki baytlık alan kaplar.
short besBin = 5000;
        
unsigned eki ile Short veri türündeki negatif sayılar es geçilir ve pozitif tarafa eklenir. Yani aralık 0 ila 65,535 olarak güncellenir.

unsigned short pozitifAltmisBin = 60000;

C'de de olduğu gibi C++'da da Türkçe karakterleri desteklememektedir.

Integer

Integer, C++ yazılım dilinde en çok tercih edilen sayısal veri türüdür. -2,147,483,648 ila 2,147,483,647 arasında sayı tutabilir. RAM'de Integer değişken türü dört baytlık alan kaplamaktadır.
int eksiYuzMilyon = -1000000;
        
unsigned eki ile Integer veri türündeki negatif sayılar es geçilir ve pozitif tarafa eklenir. Yani aralık 0 ila 4,294,967,295 olarak güncellenir.

unsigned int yuzMilyon = 1000000;

Long

Long, C++ yazılım dilinde en büyük aralığa sahip sayısal değişken türüdür.  -9,223,372,036,854,775,808 ila 9,223,372,036,854,775,807 aralığında değer alabilir. RAM'de Long değişken türü sekiz baytlık alan kaplar.
long long borc = -7685541716LL;
        
Long ifadesi kullanırken çift şekilde yani long long olarak kullanılır. Ayrıca değeri tanımlarken sol tarafına "LL" harflerini koymalıyız ki program bunun bir Long değeri olduğunu algılasın.

unsigned eki ile Long veri türündeki negatif sayılar es geçilir ve pozitif tarafa eklenir. Yani aralık 0 ila 1.8446744e+19 olarak güncellenir.

unsigned long long dunyaNufusu = 8000000000LL;

Float

Float, C++ yazılım dilinde ondalıklı sayıları belirtmek için kullanılan bir değişken türüdür. Aralığı ise 3.4 × 10^-38 ila 3.4 × 10^38 arasındaki sayıları tutabilir. RAM'de Float değişken türü dört baylık alan kaplar.
float eksiOnIkiBucuk = -12.5f;
        
Float değişken türlerine veri atarken, değerlerin sol kısma "f" işareti koyulmalıdır. Çünkü program Float bir değer olduğunu algılasını sağlar.

Double

Double, C++ yazılım dilinde en büyük ondalıklı sayı aralığına sahip değişken türüdür. Aralığı ise 4,94065645841246544E-324 ila 1,79769313486231570E+308 sayıları arasında değer alabilir.
double karmasikSayi = 2313123.34234e10;
        
Float gibi sayının sol kısmına herhangi bir ifade yazılmaz. Bunun sebebi C++ yazılım dilinde tanımlanan tüm ondalıklı sayılar Double veri türünde olarak tanımlanır.

İsimlendirme Kuralları

Pek tabi fark etmişsinizdir ki değişkenlerin isimlerini kafamıza göe koyamıyoruz. Bunun sebebi C++ yazılım dilindeki adlandırma kurallarıdır. Adlandırma kuralları bir dilde düzeni sağlar ve programın kafasının karışmasını önler. İsimlendirme kuralları normal bağlamsal yani isimin ne anlama geldiğine bakmaz(Özel ifadeler hariç). Şimdi isimlendirme kurallarını tablo şeklinde görelim.
Kural İsmi Açıklaması Doğru Kullanım Yanlış Kullanım
ASCII Kurallarına Uymak Değişken isimlendirirken sadece alfanumarik ve alt tire(_) kullanılmalıdır. karmasik_bir_sayi Karşmaşık Bir Sayı
Harf veya Alt Tire İle Başlanmalı Değişken ismi harf veya alt tire ile başlamalı. 12_sayisi sayi12
Özel Kelimeler Olmamalı Değişken ismi C programlama dili için özel yeri olan yapıların ismi olmamamlı. forDongusuIcinMiktar for

Char

Char, C++ programlama dilinde tek bir karaterlik veriyi depolamak için kullanılan bir değişken türüdür. Sayı aralığı -128 ila 127 arasında sayı alır ve RAM'de bir baytlık yer kaplar. Değer tek tırnak(') içerisine yazılır.
char birA_Harfi = 'A';
        
unsigned eki ile Char veri türündeki negatif sayılar es geçilir ve pozitif tarafa eklenir. Yani aralık 0 ila 255 olarak güncellenir.

char birA_Harfi = 65;

Char aynı zamanda sayısal değerde alabildiğini söylemiştik. ASCII adlı bir tablo sayesinde hangi sayıların hangi harfe tekabul geldiğini görebiliriz. ASCII unsigned eki ile Char ifadeleri belirtir. ASCII tablosuna göre altmış beş saysısı büyük "A" harfine denk gelmektedir.

Peki ya tek karakterlik bir yazı yerine çok karakterlik bir değeri atama yapmak isteseydik? İşte o zaman ilerki konularda daha detaylı işleceğimiz olan diziler yardımıza koşar. Dizi liste demek ve Char dizisi ile çok karakterli ifadeler yazılabilir. Çift tırnak(") içerisine yazılır. Ancak dikkat Char dizesi veya diğer adıyla String Integer veya Long'a benzemez. Bu sebeble o değişken tiplerinde yapılan işlemler bu yapıya zarar verebilir. Bu sebeble dikkatle kullanınız.

char helloWord[] = "Hello World";

Boolean

Boolean veya kısaca Bool, C'de bulunmayan fakat C++ tarafından eklenen yeni bir türdür. Evet normal şartlarda C dilinde Bool veri tipini kullanamazsınız. Çünkü tanımlı değildir. Bool veri tipine sahip değişkenlerin değerleri, açık ve bir anlamına gelen true; kapalı ve sıfır anlamına gelen false değerlerini alabilirler.
bool acik = true;
bool kapali = false;

İstersekte sayısal olarak da yazabiliriz.

bool acik = 1;
bool kapali = 0;

Aynı anlama gelmektedirler.

Auto

Auto; C++ yazılım dilindeki, değişken türlerini değere bakarak tipini belirleyen bir araçtır. Değişken tanımlarken veri tipinin girildiği kısma auto ifadesini girelim değişken tipi otomatik olarak belirlenir.
auto otomatikVeriTipiAtananDegisken = 123LL;
        
İfadeye bakılırsa bu değişkenin tipi Long değişken türüdür.

Depolama Sınıfları

C++ programlama dilinde depolama sınıfları, bir değişkenin nasıl saklancağını ve davranışlarını belirlen bir belirteçlerdir. Bu sınıfları tablo şeklinde aşağıda verilmiştir.
Ismi Açıklaması Kullanımı
Static Herkese açık hale getiren bir depolama sınıfıdır. static int herkeseAcikDegisken = 12;
Extern Dış kütüphaneleri içerisine veri yazdırabildiği ve atayabildiği değişken depolama sınıfıdır. Genellikle bir ".dll" veya ".so" dosyası bu değişkene bağlıdır ve çalışma zamanında o Extern değişkene verisini atar. extern long dllDegiskeni;
Register Register ile depolama sınıfı belirnen değişkenler standart RAM'e kayıt olmak yerine işlemcinini en hızlı veri çekip yazdığı kısım Registers kısmına depolanırlar. Ancak kullanımı sık olmamalı ve sadece hız gerektiren işlemlerde kullanılmalıdır. Çünkü İşlemcilerin Regiser kısımları RAM gibi Gigabaytlar kadar değil on altı megabayt ve iki megabayt gibi çok küçük bir alana sahiptir. Yine de C++ bizlere bu kullanımı sunmaktadır. register int enHizliVeriDepolama = 32;

Kullanıcıdan Veri Alma ve Verme

Geldik C++ programlama dilinin C'ye göre avantajlı yanına... Girdiler ve çıktılar! Bu konu C programlama dilinde çıktı için printf ve girdi için ise scanf veya gets kullanılırdı. Bazen kolay olsada aslında güvenlik açıkları çok fazlaydır. Bu sebeble C++ geliştiricileri girdileri G/Ç[Giriş/Çıkış(Input/Output)] kütüphanesinde Cin ve Cout yapılarına dönüştürmüşledir.
Cout yani C out, konsola çıktı vermemizi sağlar. Çıktı vermek için ise "Operatörler" daha detaylı işliceğimiz olna sola kaydırma operatörü aracılığıyla sağlar. Sola kaydırma kısaca sağdaki verileri soldaki kısma aktarır. Böylelikle değerle çıktı verilebilir hale gelir.
Bir Integer değerini Cout ile konsola yazdıralım.

#include <iostream>

using namespace std;

int main()
{
    int birSayi = 14;

    cout << birSayi << endl;

    return 0;
}

Derleyip çalıştırdığımızda çıktı "14" olucaktır. Gördüğünüz gibi bir değişkenin değerini bu şekilde yazdırabiliyoruz.
Girdi almak istersekde Cin yani C in'i kullanırız. Cin, kullanıcının girdiği değeri sağ kaydırma operatörü ile tanımlanmak istenen değişkenin değerine atar. Aynı şekilde sağ kaydırma operatörünü "Operatörler" sayfasında daha detaylı işleceğiz. Girilen ifade Cin tarafından Char dizisi olarak dönmektedir.
Şimdi ise girilen bir yazı ifadesini Cin ile alarak Cout'la konsola tekrar yazdırsın.

#include <iostream>

using namespace std;

int main()
{
    cout << "Bir ifade giriniz: ";

    char inp[256];

    cin >> inp;

    cout << "Girilen ifade '" << inp << "'" << endl;

    return 0;
}

Çıktısı ise;
Alınan veri ise ilk kelime olarak alır.

Değişkenleri Türlerini Dönüştürme

Artık değişken tanımlamayı ve kullanıcıdan girdi almayı biliyoruz. Fakat gelen veri Integer ve ihtiyacımız olan veri türü Float ise? O zaman hangi işleme başvurmalıyız? Bu gibi durumlarda değişken tiplerini dönüştürme yapısı gelir. Şimdi veri türlerinin nasıl dönüştürüldüğünü inceleyelim.

Integer → Float

Elimizdeki Integer değerini Float'a dönüştürmek istiyorsak, Integer değerinin başına parantez içerisinde "float" yazarız. Bu yapı bize Integer değermizi Float değişken türüne dönüştürmemizi sağlar.
Şimdi bir Integer verisini Float'a dönüştürelim ve bu işlemi Cout ile çıktı halinde konsola yazdıralım.

#include 

using namespace std;

int main()
{
    int integerBirDeger = 45;

    float integerDegeriFloatHali = (float) integerBirDeger + 0.5f;
    
    cout << "'" << integerBirDeger << "'" << "-> '" << integerDegeriFloatHali << "'" << endl;

    return 0;
}

Float değeri gözleyebilmek için dönüşmüş Integer'a 0.5 değeri ekledim ve konsola yazdırdım.

Çıktısı ise "'45'-> '45.5'" olucaktır.

Float → Integer

Bu durumda ise parantezlerin içerisine "int" yazılır ve değer Float'dan Integer'a dönüşmüş olur. Ayrıca Integer'a dönüştüğü için ise noktadan sonraki yok olur.
Aynı işlemin bu sefer tersini gerçekleştirelim.

#include <iostream>

using namespace std;

int main()
{
    float floatBirDeger = 12.45f;

    int floatDegeriIntegerHali = (int) floatBirDeger;
    
    cout << "'" << floatBirDeger << "'" << " -> '" << floatDegeriIntegerHali << "'" << endl;

    return 0;
}

Çıktısı ise "'12.45' -> '12'" olucaktır.

Sabit Değişkenler

Sabit değişkenler, bir kere veri atanıp daha sonra bir daha verisi değiştirilemeyen değişkenlerdir.
C++ programlama dilinde kullanılabilir üç adet sabit değişken türü vardır. Mantıkları aynıdır fakat depolanma biçimleri ve RAM'in hangi bölgesine kayıt edildiği farklıdır.

Const

Const, en çok kullanılan ikinci sabit değişken türüdür. Depolandığı kısım ise RAM'in sabir veriler adlı bir bölümüne kayıt edilir. Bu sayede sürekli değişiklik göstermesi gerek normal RAM bölgesinde depolanmaz ve yer tasarufu sağlanır.
Pi sayısını Float cinsinden Const ile tanımlayalım ve ekrana bastıralım.

#include <iostream>

using namespace std;

int main()
{
    const float pi = 3.14f;

    cout << "Pi sayisinin degeri yaklasik olarak ; " << pi << endl;

    return 0;
}

Çıktısı ise "Pi sayisinin degeri yaklasik olarak ; 3.14" şeklinde olucaktır.

Constexpr

Bu sabitlik türü ise değerlerin çalışma zamanında atanması yerine derleme zamanında atanarak performans ve hız kazandırır. Depolandığı kısım Const ile aynıdır.
Aynı örneği bu sefer Constexpr ile yapalım.

#include <iostream>

using namespace std;

int main()
{
    constexpr float pi = 3.14f;

    cout << "Pi sayisinin degeri yaklasik olarak ; " << pi << endl;

    return 0;
}

Değişen bir durum yoktur fakat performans açısından en verimli olan sabitde Constexpr'dir. Çıktısı aynı olucaktır tabi...

Define

Define, Const ve Constexpr'den farklı olarak bir değişken teknik olarak değildir. Bir makrodur ve RAM'de depolanmaz. Depolanmadığı için de değiştirilmez özelliği vardır. Bu sebeble sabit değişkenlerde sıklıkla kullanılır. Define veya diğer adıyla makro, zorunlu değildir fakat kullanımı sırasında değişkenin isminin harfleri büyük harf ile yazılması tavsiye edilir. Çünkü dediğimiz gibi Define teknik olarak bir değişken değildir. Define'i kullanırken başına diyez(#) işareti konulur. Bu durum "Önişlemciler" sayfasında daha detaylı işlenicektir.
Aynı örneğimizi Define ile yapalım.

#include <iostream>

using namespace std;

#define PI 3.14f

int main()
{
    cout << "Pi sayisinin degeri yaklasik olarak ; " << PI << endl;

    return 0;
}

Çıktısı diğer örneklerle aynıdır.

Typedef

Typdef, değişken türlerini bir nevi bir değişken gibi atamamızı sağlayan bir yapıdır.
Mesela biz Integer ifadesini "int" kelimesi ile ifade ediyoruz, ama başka şekilde ifade etmek istersek? Mesela "int" yernine "tamSayi" gibi... İşte bunun için Typdef kullanılır.
 Şimdiki örneğimizin Float değişken tipini Typedef ile "ondalikli" olarak değiştirelim ve Constexpr ile de sabit bir değişen olduğunu belirtelimç
            
#include <iostream>

using namespace std;

typedef float ondakilki;

int main()
{
    constexpr ondakilki pi = 3.14f;

    cout << "Pi sayisinin degeri yaklasik olarak ; " << pi << endl;

    return 0;    
}

Artık Float değişken tipimizin ismi "ondalikli" olarak bu kodda atanmış oldu. Çıktı değişmicektir.

Using

İlk kod örneğimizde gördüğümüz gibi bir kütüphanenin içerisindeki isim alanlarını koda dahil etmek ve kolay kullanım için Using ile dahil edilir.
Ya Using ile isim alanı katılımını yapmasaydık? Şimdi bu durumda kodlarımızı yazalım.

#include <iostream>

typedef float ondakilki;

int main()
{
    constexpr ondakilki pi = 3.14f;

    std::cout << "Pi sayisinin degeri yaklasik olarak ; " << pi << std::endl;

    return 0;    
}

Gördüğünüz gibi Cout ve Endl'nin sol taraflarına Std ifadesiyle iki adet iki nokta geldi. Bu durum aslında bişi şunu açıklar; Cout ve Endl elemanları Std adlı isim alanın içerisinde. Using, kolay kodlama için elzem niteliğindedir.

Çıktı değişmicektir.
